J'apprends à coder en HTML

Le mieux est de commencer par du HTML car c'est un langage simple.

Mais attention, simple ne veut pas forcément dire facile.

Je conseille de trouver de bons tutos sur internet.

Et surtout de ne pas oublier de lire la documentation officielle de HTML sur le site Mozilla Developper Network.

J'apprends à coder en CSS

Mais très vite le besoin d'aller plus loin se fait ressentir.

C'est là qu'il devient intéressant d'apprendre le CSS car ce langage permet de donner vie à votre contenu en y ajoutant du style.

De nouveau il y a la documentation officielle de CSS sur le site Mozilla Developper Network.

Peut-être que je me mettrai aussi au Javascript

Là on passe à un autre niveau.

Je vais tranquilement découvrir HMTL et CSS mais je sens que vais vouloir attaquer ce gros morceau bientôt !

Une fois de plus il y a la documentation officielle de Javascript sur le site Mozilla Developper Network.

Fait avec amour en HTML et CSS.
